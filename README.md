# divipo
Division by increasing power order of binary cyclic code
This polynomial division is used in the context of decoding binary cyclic codes under a certain validity condition (tdeg(h(x))<n).
For more information, please refer to my article.
Here is the link: https://rjm-cs.utcb.ro/wp-content/uploads/2026/03/RJM-CS_online_first_2026_paper_2_Andrianalisefa.pdf

Explanation:
- n: code length
- d: minimum distance
- g: generator polynomial
- a: received word whose error has weight less than or equal to t
